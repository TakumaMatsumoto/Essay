\section{非線形光学効果}
\subsection{非線形光学効果の概要}
レーザー出現以前に知られていた弱い光によっておこる直進，反射，屈折，回折，干渉，吸収，発光，散乱といった様々な光学現象はすべて線形光学現象と呼ばれる．ここで線形であることは光に対する物質の応答が線形すなわち光電場に比例しているということを意味している．これは，光の電界によって誘起される分極によるものである．光の強度が小さい範囲では物質の分極が光の電界の大きさに比例するが，光の強度が大きくなると分極が光の電界に比例しなくなり，非線形性が見られるようになる．このとき発生する現象が非線形光学現象である{\cite{non_linear}}．

\subsection{非線形屈折率変化と非線形現象}
非線形屈折率変化とは光の強度に応じて屈折率が変化する現象で，分極の非線形性が原因となって引き起こされる．光強度を$I$，線形な範囲での屈折率を$n_0$，非線形屈折率係数を$n_2$とすると屈折率は，
\begin{equation}
  n = n_{0} + n_{2}I
\end{equation}
と表すことができる．

非線形屈折率変化によって引き起こされる現象には，自分自身の光の強度により位相が変化する自己位相変調(SPM:Self Phase Modulation)，他の光の強度により位相が変化する相互位相変調(XPM:Cross Phase Modulation)が存在する．

\section{半導体光増幅器(SOA:Semiconductor Optical Amplifier)}
SOAは半導体を用いた光増幅器である{\cite{optic}}．SOAに外部から励起させた半導体に光を照射すると，入射光の強さに比例した位相も周波数も同じ光を放出する，誘導放出という現象を利用し，増幅した光を出力する．

\subsection{SOAの基本構造}
図{\ref{fig:SOA}}にSOAの簡易構造を示す．SOAはp型とn型の異なる原子配列を持った半導体に活性層が挟み込まれた構造となっており，この活性層で光が増幅される．活性層には単一の元素からなる，不純物を含まない真性半導体が用いられるが，p型とn型の半導体には，キャリア（自由電子や正孔のように電荷を運ぶもの）を増やし電気伝導率を上げるために真性半導体に不純物を添加した不純物半導体を用いる．このうち，自由電子キャリアを増加させたものをn型半導体，自由正孔キャリアを増加させたものをp型半導体と言う．

\subsection{SOAにおける光の増幅原理}
二つのエネルギー準位$E_{1}，E_{2}(E_{1}<E_{2})$をもつ原子系を考える．一般に高いエネルギー準位（準位2）にあるキャリアは，ある確率で低いエネルギー準位（準位1）に遷移する．この過程を図(左)に示す．この過程は媒質に入射する光に左右されないため，自然放出と呼ばれる．\\
一方，媒質に光が入射すると，準位1にあるキャリアが準位2に遷移する誘導吸収と，準位2にあるキャリアが準位1に遷移する誘導放出が起こる．これは媒質に入射する光によって起こるため，誘導遷移過程と呼ばれる．誘導吸収過程を図(中央)に，誘導放出過程を図(右)に示す．\\
ここで，準位1におけるキャリアの数を$N_{1}$，準位2におけるキャリアの数を$N_{2}$とすると，平衡状態では$N_{1}<N_{2}$となる．SOAにおいては，電流によってキャリアを注入することによって，反転分布$N_{1}>N_{2}$を形成する．反転分布においては，誘導吸収されるキャリアの数より，誘導放出されるキャリアの数を大きくなるため，光が増幅される．

\section{量子井戸(QW:Quantum Well)}
QWとは2種類の半導体薄膜が交互に積み重なった構造のことであり，キャリアを2次元方向に閉じ込めることができる．図{\ref{fig:qw_energy}}にQWのバンド図を示す．QWでは電子・正孔の取りうるエネルギーが量子化され，伝導帯・価電子帯に階段状のエネルギー準位が形成される．これに伴い，スペクトル幅が狭くなり，光学利得が改善される．光ファイバ通信で用いられる波長領域では，GaInAsP，AlGaInAs系材料を用いて，大きいバンドギャップ材料が小さいバンドギャップ材料を挟む量子井戸構造が用いられる．この構造では，キャリアが低いバンドギャップに落ち込み，光学遷移が小さいバンドギャップ材料の伝導帯，価電子帯の量子準位間で生じ，吸収，発光の効率を高くすることができる．

\section{量子ドット(QD:Quantum Dot)}
QDは複数の半導体原子から構成されており，キャリアを3次元方向に閉じ込めることができる．図{\ref{fig:qd_energy}}にQDのバンド図を示す．QDはQWと比較して，より多くのエネルギー準位を形成し，QWよりも吸収，発光の効率を高くすることができる．QDはバルク結晶上に薄膜を塗布し，薄膜を形成する原子とバルク結晶の相互作用により薄膜上に形成させるStranski-Krastanovモードと呼ばれる手法で形成される．

\section{量子ドット半導体光増幅器(QD-SOA)}

\subsection{QD各層のエネルギー準位，密度}
QDはどのような製造過程においても大きさを制御することができないため，エネルギー準位にばらつきが起こる{\cite{qd_dispersion}}．各層のエネルギー準位の密度のばらつきは正規分布に従うものとし，サンプル数を$2M+1$として量子化を行うと伝導帯，価電子帯のj番目のエネルギー準位は最も密度が高い標本番号を$M$とするとそれぞれ
\begin{eqnarray}
	E^{c}_{u(e,g),j} & = & E^{c}_{u(e,g),M} + \left[ j - \left( M+1 \right) \right] \times r_{c} \Delta E \\
	E^{v}_{u(e,g),j} & = & E^{v}_{u(e,g),M} - \left[ j - \left( M+1 \right) \right] \times r_{v} \Delta E
\end{eqnarray}
と表される．図{\ref{fig:energy_dist}}にQD各層のエネルギー準位と密度の関係を示す．図{\ref{fig:energy_dist}}のように各層のエネルギー準位の密度は正規分布に従うものとし，このときj番目の割合は平均を$E^{c(v)}_{g,M}$分散を$\sigma_{in,c(v)} = \Gamma^{c(v)}_{ih} / 2.356$とした正規分布となり，
\begin{equation}
  G^{c(v)}_{j} = \frac{1}{ \sqrt{2 \pi \sigma^{2}_{in,c(v)}}} \exp\left[-\frac{ \left( E^{c(v)}_{g,j} - E^{c(v)}_{g,max} \right)^{2} }{ 2 \sigma^{2}_{in,c(v)} } \right] \times r_{c(v)} \Delta E
\end{equation}
と表すことができる．

\subsection{レート方程式}
QD-SOAの入出力特性を解析するためには，QD-SOAに光を入射した際のキャリアの増減に起因する利得の変化，およびそれに伴う光の振幅と位相の変化を計算する必要がある．半導体発光デバイスにおけるキャリアの時間変化はレート方程式と呼ばれる微分方程式で記述できることが知られている{\cite{qd_soa_rate_equation}}．本研究に用いるレート方程式を式に示す．

\begin{eqnarray}
  \label{eq:qd_qw}
  \frac{ {\partial}f^{c(v)}_{w} }{ {\partial}t } & = & 
	\frac{I}{qN^{c(v)}_{W}} - 
	\frac{ \sqrt{ f^{c}_{w} f^{v}_{w} } }{ \hat{\tau}^{c(v)}_{wr} } + 
	\frac{ D^{c(v)}_{u} }{ D^{c(v)}_{w} } \sum_{j=1}^{2M+1} G^{c(v)}_{j} \nonumber\\ &&
	\times
	\left(
		\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{uw,j} }	\left(1 - f^{c(v)}_{w} \right)
		\frac{ f^{c(v)}_{w}  }{ \hat{\tau}^{c(v)}_{wu}  } 	\left(1 - f^{c(v)}_{u,j}\right)
	\right)
  \\
  \label{eq:qd_us}
  \frac{ {\partial}f^{c(v)}_{u,j} }{ {\partial}t } & = & 
	\frac{ f^{c(v)}_{w}  }{ \hat{\tau}^{c(v)}_{wu}  }	\left(1 - f^{c(v)}_{u,j}\right) -
	\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{uw,j} }	\left(1 - f^{c(v)}_{w} \right) \nonumber\\ &&
	+ \frac{ D^{c(v)}_{e} }{ D^{c(v)}_{u} } \left(
		\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eu} } \left(1 - f^{c(v)}_{u,j}\right)
		\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ue} } \left(1 - f^{c(v)}_{e,j}\right)
	\right)  \nonumber\\ &&
	+ \frac{ D^{c(v)}_{g} }{ D^{c(v)}_{u} } \left(
		\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{gu} } \left(1 - f^{c(v)}_{u,j}\right)
		\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ug} } \left(1 - f^{c(v)}_{g,j}\right)
	\right)  \nonumber\\ &&
	- \frac{ \sqrt{ f^{c}_{u,j} f^{v}_{u,j} } }{ \hat{\tau}^{c(v)}_{ur} }
  \\
  \label{eq:qd_es}
  \frac{ {\partial}f^{c(v)}_{e,j} }{ {\partial}t } & = & 
	\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ue} }	\left(1 - f^{c(v)}_{e,j}\right) -
	\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eu} }	\left(1 - f^{c(v)}_{u,j}\right) \nonumber\\ &&
	+ \frac{ D^{c(v)}_{g} }{ D^{c(v)}_{e} } \left(
		\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{ge} } \left(1 - f^{c(v)}_{e,j}\right)
		\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eg} } \left(1 - f^{c(v)}_{g,j}\right)
	\right)  \nonumber\\ &&
	- \frac{ \sqrt{ f^{c}_{e,j} f^{v}_{e,j} } }{ \hat{\tau}_{dr} } - \frac{\Gamma}{N^{c(v)}_{E,j}} \nonumber\\ &&
	\times
	\sum_{k} \frac{ P_{k,in}g^{e}_{jk} \bigl[e^{( [ \Gamma g(t,E_k) - \alpha_{i} ] L_{ca})} - 1 \bigr]}{E_k\bigl[ \Gamma g \left( t,E_k \right) - \alpha_{i} \bigr]} \nonumber\\ &&
	\times
	\left( f^{c}_{e,j} + f^{v}_{e,j} - 1 \right)
  \\
  \label{eq:qd_gs}
  \frac{ {\partial}f^{c(v)}_{g,j} }{ {\partial}t } & = & 
	\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ug} }	\left(1 - f^{c(v)}_{g,j}\right) -
	\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{gu} }	\left(1 - f^{c(v)}_{u,j}\right) \nonumber\\ &&
	\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eg} } \left(1 - f^{c(v)}_{g,j}\right) -
	\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{ge} } \left(1 - f^{c(v)}_{e,j}\right) \nonumber\\ &&
	- \frac{ \sqrt{ f^{c}_{g,j} f^{v}_{g,j} } }{ \hat{\tau}_{dr} } - \frac{\Gamma}{N^{c(v)}_{G,j}} \nonumber\\ &&
	\times
	\sum_{k} \frac{ P_{k,in}g^{g}_{jk} \bigl[e^{( [ \Gamma g(t,E_k) - \alpha_{i} ] L_{ca})} - 1 \bigr]}{E_k\bigl[ \Gamma g \left( t,E_k \right) - \alpha_{i} \bigr]} \nonumber\\ &&
	\times
	\left( f^{c}_{g,j} + f^{v}_{g,j} - 1 \right)
\end{eqnarray}

({\ref{eq:qd_qw}})式から({\ref{eq:qd_gs}})式の左辺は活性層の各層でのキャリア占有率の時間微分を表している．ここで，各式の詳細について以下に述べる．
式({\ref{eq:qd_qw}})は微小時間におけるQWにおけるキャリア占有率の挙動である．右辺各項について説明する．
\begin{description}

  \item[式({\ref{eq:qd_qw}})第3項 $\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{uw,j} }	\left(1 - f^{c(v)}_{w} \right)$]\mbox{}\\

\subsection{QD-SOAによる利得}
QD-SOAの利得$g(t,E_k)$は
\begin{equation}
  g(t,E_k) = \sum_{l=e,g}\sum_{j=1}^{2M+1} g^{l}_{j,k}(E_k)\bigl[f^{c}_{l,j}+f^{v}_{l,j}-1\bigr]
\end{equation}
で表される．これは，利得$g$がES，GSのキャリア占有率に依存することを示している．\\
また，j番目ES，GS層が波長kの光子に与える利得$g^{e(g)}_{j,k}$は
\begin{equation}
  g^{e(g)}_{j,k} = \frac{ \pi \hbar q^{2} N_{D} D_{e(g)} }{ c n_{r} \varepsilon_{0} m^{2}_{0} E_{k} } \left|M_{env}\right|^{2} \left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> L^{e(g)}_{j} G^{s}_{j}
\end{equation}
で表される．ここで，$L^{e(g)}_{j}$，$G^{s}_{j}$はそれぞれ光子エネルギーの均一広がり，j番目QD層の放射再結合の割合を表しており，
\begin{eqnarray}
  L^{e(g)}_{j}  & = & \frac{1}{\pi} \frac{ \Gamma_{h} / 2 }{\left(E^{s}_{e(g),j} - E_k\right)^{2} + \left(\Gamma_{h}/2\right)^{2}} \\
  G^{s}_{j}  & = & \frac{1}{\sqrt{2 \pi \sigma^{2}_{in,s}}} \exp \left[-\frac{\left(E^{s}_{g,j}-E^{s}_{g,M}\right)^{2}}{2\sigma^{2}_{in,s}}\right] \Delta E
\end{eqnarray}
と表される．また$E^{s}_{u(e,g),j}$はQD各層における光子遷移エネルギーであり，
\begin{equation}
  E^{s}_{u(e,g),j} = E^{c}_{u(e,g),j} - E^{v}_{u(e,g),j}
\end{equation}
となる．

\subsection{QD-SOAによる位相変化}
QD-SOAはバンド充填効果，自由電子吸収効果により屈折率変化が起き，それにより光の位相変化が起きる．以下にバンド充填効果による屈折率変化の式を示す．
\begin{eqnarray}
  \Delta n^{w}_{ib} (t,E_k) & = & \sum_{j=1}^{2M+1} \frac{ -\Gamma_{w} \pi \hbar^{2} q^{2} N_{D}}{2 n_{r} \varepsilon_{0} m^{2}_{0} E^{2}_{k}} \left|M_{env}\right|^{2} \left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> \nonumber\\ &&
  \times \frac{ D^{c}_{w} D^{v}_{w} }{D^{c}_{w} + D^{v}_{w}} G^{s}_{j}B_{w}\left[f^{c}_{w}+f^{v}_{w}-1\right]
  \\
  \Delta n^{u}_{ib} (t,E_k) & = & \sum_{j=1}^{2M+1} \frac{ -\Gamma \pi \hbar^{2} q^{2} N_{D}}{2 n_{r} \varepsilon_{0} m^{2}_{0} E^{2}_{k}} \left|M_{env}\right|^{2} \left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> \nonumber\\ &&
  \times \frac{ D^{c}_{u} D^{v}_{u} }{D^{c}_{u} + D^{v}_{u}} G^{s}_{j}B_{u,j}\left[f^{c}_{u,j}+f^{v}_{u,j}-1\right]
  \\
  \Delta n^{e(g)}_{ib} (t,E_k) & = & \sum_{j=1}^{2M+1} \frac{ -\Gamma \pi \hbar^{2} q^{2} N_{D} D_{e(g)} }{2 n_{r} \varepsilon_{0} m^{2}_{0} E^{2}_{k}} \left|M_{env}\right|^{2} \left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> \nonumber\\ &&
  \times G^{s}_{j}B_{e(g),j}\left[f^{c}_{e(g),j}+f^{v}_{e(g),j}-1\right]
\end{eqnarray}
ここで$B_{w}$，$B_{u(e,g),j}$は屈折率の均一広がりを表し，以下の式で与えられる．
\begin{eqnarray}
  B_{w} & = & \frac{\left(E^{s}_{w} - E_{k}\right) / \pi}{\left(E^{s}_{w} - E_{k}\right)^{2} + \left(\Gamma_{h} / 2\right)^{2}} \\
  B_{u(e,g),j} & = & \frac{\left(E^{s}_{u(e,g),j} - E_{k}\right) / \pi}{\left(E^{s}_{u(e,g),j} - E_{k}\right)^{2} + \left(\Gamma_{h} / 2\right)^{2}}
\end{eqnarray}
自由電子吸収効果による屈折率変化の式は以下のようになる．
\begin{eqnarray}
  \Delta n^{w}_{fc} (t,E_k) & = & - \frac{ \Gamma_{w} \hbar^{2} q^{2} }{ 2 V_{w} n_{r} \varepsilon_{0} E^{2}_{k} } 
  \left(
	\frac{ N^{c}_{W} }{ m^{*c}_{w} } f^{c}_{w} + 
	\frac{ N^{v}_{W} }{ m^{*v}_{w} } f^{v}_{w}
  \right) 
  \\
  \Delta n^{u(e,g)}_{fc} (t,E_k) & = & - \frac{ \Gamma \hbar^{2} q^{2} }{ 2 V_{d} n_{r} \varepsilon_{0} E^{2}_{k} } \nonumber\\ &&
  \left( \sum_{j=1}^{2M+1} 
	  \frac{ f^{c}_{u(e,g),j} N^{c}_{U(E,G),j} }{ m^{*c}_{d} } 
	+\frac{ f^{v}_{u(e,g),j} N^{v}_{U(E,G),j} }{ m^{*v}_{d} }
  \right)
\end{eqnarray}
バンド充填効果，自由電子吸収効果による屈折率変化量の和を$\Delta n(t,E_k)$とすると位相変化量は
\begin{equation}
  \Delta \phi (t,E_k) = -\frac{2 \pi L_{ca}}{\lambda_{k}} \Delta n(t,E_k)
\end{equation}
と表される．

\subsection{QD-SOA内の光の伝搬方程式}
QD-SOA内の光の伝搬方程式は
\begin{equation}
  \label{eq:denpan}
  \frac{{\partial}E}{{\partial}z} = \frac{E}{2}{-\alpha_{int} + (1-j\alpha_{LEF})g_{\max}(2f-1)}
\end{equation}
で表せる．$z$はQD-SOA内の光の伝搬方向の位置，$\alpha_{int}$は損失係数，$\alpha_{LEF}$は線幅増大係数，$j$は虚数単位を表している．QD-SOAにおける位相変化量を$\phi$とすると，式({\ref{eq:denpan}})より
\begin{equation}
  \label{eq:phi_diff}
  \frac{{\partial}\varphi}{{\partial}z} = -\frac{1}{2}\alpha_{LEF}g_{\max}(2f-1)
\end{equation}
と表せる．式({\ref{eq:phi_diff}})より，QD-SOAにおける位相の変化量は線幅増大係数，最大利得，GSにおけるキャリア占有率に依存することが分かる．つまり，QD-SOAのパラメータである線幅増大係数，最大利得を変化させることで，位相変化量の調整が可能である．

\subsection{非線形状態空間法(NSSM:Nonlinear State Space Method)}
QD-SOA内における光電界はキャリア密度や時間，位置に依存して絶えず変化する．また，QD-SOAはデバイス長が短いものでも数$100{\mu}m$程度あるため，入力時の光電界から直接，出力時の光電界を求めることができない．そこで，{\cite{NSSM}}ではNSSMを用いてQD-SOAの入出力特性の解析をしている．

\section{全光XORゲート}

\section{消光比}
アイダイアグラムを数値的に評価する指標として消光比(ER:Extinction Ratio)が用いられ，これはシステムを考える上での一つの性能指標となる．消光比(ER）は下式によって表される．
\begin{equation}
  ER(dB) = 10 \log \frac{P_{\min}^{1}}{P_{\max}^{0}}
\end{equation}
ここで，$P_{\min}^{1}$および$P_{\max}^{0}$は図{\ref{fig:extinction}}に対応しており，それぞれ出力信号光``1''に対応する最小光強度，出力信号光``0''に対応する最大光強度を表す．
