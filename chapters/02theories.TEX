\section{非線形光学効果}
\subsection{非線形光学効果の概要}
レーザー出現以前に知られていた弱い光によっておこる直進，反射，屈折，回折，干渉，吸収，発光，散乱といった様々な光学現象はすべて線形光学現象と呼ばれる．ここで線形であることは光に対する物質の応答が線形すなわち光電場に比例しているということを意味している．これは，光の電界によって誘起される分極によるものである．光の強度が小さい範囲では物質の分極が光の電界の大きさに比例するが，光の強度が大きくなると分極が光の電界に比例しなくなり，非線形性が見られるようになる．このとき発生する現象が非線形光学現象である{\cite{non_linear}}．

\subsection{非線形屈折率変化と非線形現象}
非線形屈折率変化とは光の強度に応じて屈折率が変化する現象で，分極の非線形性が原因となって引き起こされる．光強度を$I$，線形な範囲での屈折率を$n_0$，非線形屈折率係数を$n_2$とすると屈折率は，
\begin{equation}
  n = n_{0} + n_{2}I
\end{equation}
と表すことができる．

非線形屈折率変化によって引き起こされる現象には，自分自身の光の強度により位相が変化する自己位相変調(SPM:Self Phase Modulation)，他の光の強度により位相が変化する相互位相変調(XPM:Cross Phase Modulation)が存在する．

\section{半導体光増幅器(SOA:Semiconductor Optical Amplifier)}
SOAは半導体のエネルギーバンドギャップを利用した光増幅器である{\cite{optic}}．SOA内のキャリアが励起している状態でSOAに光を入力すると位相と周波数が等しい光がキャリアから放出され，活性層内で光が増幅される．

\subsection{SOAの基本構造}
図{\ref{fig:model_of_soa}}にSOAの簡易構造を示す．SOAはp型とn型の異なる原子配列を持った半導体に活性層が挟み込まれた構造となっており，この活性層で光が増幅される．活性層には単一の元素からなる，不純物を含まない真性半導体が用いられるが，p型とn型の半導体には，キャリア（自由電子や正孔のように電荷を運ぶもの）を増やし電気伝導率を上げるために真性半導体に不純物を添加した不純物半導体を用いる．このうち，自由電子キャリアを増加させたものをn型半導体，自由正孔キャリアを増加させたものをp型半導体と言う．

\subsection{SOAにおける光の増幅原理}
二つのエネルギー準位$E_{1}，E_{2}(E_{1}<E_{2})$をもつ原子系を考える．一般に高いエネルギー準位（準位2）にあるキャリアは，ある確率で低いエネルギー準位（準位1）に遷移する．この過程を図(左)に示す．この過程は媒質に入射する光に左右されないため，自然放出と呼ばれる．\\
一方，媒質に光が入射すると，準位1にあるキャリアが準位2に遷移する誘導吸収と，準位2にあるキャリアが準位1に遷移する誘導放出が起こる．これは媒質に入射する光によって起こるため，誘導遷移過程と呼ばれる．誘導吸収過程を図(中央)に，誘導放出過程を図(右)に示す．\\
ここで，準位1におけるキャリアの数を$N_{1}$，準位2におけるキャリアの数を$N_{2}$とすると，平衡状態では$N_{1}>N_{2}$となる．SOAにおいては，電流によってキャリアを注入することによって，反転分布$N_{1}<N_{2}$を形成する．反転分布においては，誘導吸収されるキャリアの数より，誘導放出されるキャリアの数を大きくなるため，光が増幅される．

\section{量子井戸(QW:Quantum Well)}
QWとは2種類の半導体薄膜が交互に積み重なった構造のことであり，キャリアを2次元方向に閉じ込めることができる．図{\ref{fig:qw_energy}}にQWのエネルギーバンド図を示す．QWでは伝導帯・価電子帯に階段状のエネルギー準位が形成される．これに伴いスペクトル幅が狭くなり，光学利得が改善される．Bulk部にはQW部よりもエネルギーバンドギャップが大きいガリウムひ素(GaAs)が用いられる．QW部に用いられる材質は使用する波長帯域によって異なり，1550nm付近の波長の光を増幅するにはひ化りん化インジウムガリウム(GaInAsP)が用いられ，1300nm付近の波長の光を増幅するにはインジウムガリウムヒ化物(InGaAs)が用いられる{\cite{theory_of_qdsoa}}．量子井戸構造はQW部がBulk部に挟まれる構造であり，キャリアはバンドギャップの小さいQW部に集中する．エネルギー準位の遷移はエネルギーバンドギャップの小さい材質の伝導帯，価電子帯の準位間で生じるため吸収，発光の効率を高くすることができる．

\section{量子ドット(QD:Quantum Dot)}
QDは複数の半導体原子から構成されており，キャリアを3次元方向に閉じ込めることができる．QDは半径の大きさによってエネルギーバンドギャップが異なり，図{\ref{fig:qd_energy}}のように半径が小さいほどエネルギーバンドギャップが大きくなる．QDはQW以上に小さいバンドギャップを形成するためQWよりも吸収，発光の効率を高くすることができる．QDはStranski-Krastanovモードと呼ばれる手法で形成する方法が一般的であり，多くの材料について行われてきている{\cite{qd_book}}．これはGaAsの基盤の上にIn原子とAs原子をぶつけてGaAs基板上に薄い膜を生成し、膜が厚くなるとInAsが自然と凝集してQDが形成されるという現象を利用している．

\subsection{QD各層のエネルギー準位，存在確率}
QDはどのような製造過程においても大きさを制御することができないため，エネルギー準位にばらつきが生じる{\cite{qd_dispersion}}．各層のエネルギー準位の存在確率は正規分布に従うものとし，サンプル数を$2M+1$として量子化を行うと伝導帯，価電子帯のj番目のエネルギー準位は最も存在確率が高いサンプル番号を$M+1$とするとそれぞれ
\begin{eqnarray}
	E^{c}_{u(e,g),j} & = & E^{c}_{u(e,g),M} + \left[ j - \left( M+1 \right) \right] \times r_{c} \Delta E \\
	E^{v}_{u(e,g),j} & = & E^{v}_{u(e,g),M} - \left[ j - \left( M+1 \right) \right] \times r_{v} \Delta E
\end{eqnarray}
と表される．図{\ref{fig:energy_dist}}にQD各層のエネルギー準位と存在確率の関係を示す．図{\ref{fig:energy_dist}}のように各層のエネルギー準位の存在確率は正規分布に従うものとし，このときサンプル番号j番目のエネルギーを放出するQDが存在する確率は平均エネルギーを$E^{c(v)}_{g,M}$，分散を$\sigma_{in,c(v)} = \Gamma^{c(v)}_{ih} / 2.356$とした正規分布となり，
\begin{equation}
  G^{c(v)}_{j} = \frac{1}{ \sqrt{2 \pi \sigma^{2}_{in,c(v)}}} \exp\left[-\frac{ \left( E^{c(v)}_{g,j} - E^{c(v)}_{g,M+1} \right)^{2} }{ 2 \sigma^{2}_{in,c(v)} } \right] \times r_{c(v)} \Delta E
\end{equation}
と表すことができる．

\subsection{キャリア緩和時間，脱出時間}
キャリア緩和時間とはキャリアがエネルギーバンドギャップの高いQD層から低いQD層へ移るまでに要する時間のことであり，キャリア脱出時間とはエネルギーバンドギャップの低いQD層から高いQD層へ移るまでに要する時間を表す．そのため，キャリア緩和時間がキャリア脱出時間よりも短いほどGS層にキャリアが集まりやすくなり，効率よく光を増幅させることができる．キャリア脱出時間は釣り合いの原理を用いてキャリア緩和時間，及び縮退係数から求めることができる{\cite{qd_dispersion,on_the_noise_properties,qd_low_noise}}．US,ES,GS間のキャリア脱出時間は2つの層をy層，x層とすると
\begin{equation}
  \label{eq:cb_carrier_escape_time}
  \tau^{c(v)}_{yx} = 
	f^{c(v)}_{D} \tau^{c(v)}_{xy} 
	\frac{D_{y}}{D_{x}} 
	\exp\left(\frac
	{\left| E^{c(v)}_{x} - E^{c(v)}_{y} \right|}
	{k_{B}T}\right)
\end{equation}
より求めることができる．また，US層からQW層へのキャリア脱出時間は
\begin{equation}
  \label{eq:cb_carrier_escape_time}
  \tau^{c(v)}_{uw,j} = 
	\tau^{c(v)}_{wu} 
	\exp\left(\frac
	{\left| E^{c(v)}_{u,j} - E^{c(v)}_{QW} \right|}
	{k_{B}T}\right)
\end{equation}
と求めることができる．

\subsection{QDSOAのレート方程式}
QDSOAの入出力特性を解析するためには，QDSOAに光を入射した際のキャリアの増減に起因する利得の変化，およびそれに伴う光の振幅と位相の変化を計算する必要がある．半導体発光デバイスにおけるキャリアの時間変化はレート方程式と呼ばれる微分方程式で記述できることが知られている{\cite{qd_soa_rate_equation}}．本研究に用いるQDSOAのレート方程式を({\ref{eq:qd_qw}})式から({\ref{eq:qd_gs}})式に示す．

\begin{eqnarray}
  \label{eq:qd_qw}
  \frac{ {\partial}f^{c(v)}_{w} }{ {\partial}t } & = & 
	\frac{I}{qN^{c(v)}_{W}} - 
	\frac{ \sqrt{ f^{c}_{w} f^{v}_{w} } }{ \hat{\tau}^{c(v)}_{wr} } + 
	\frac{ D^{c(v)}_{u} }{ D^{c(v)}_{w} } \sum_{j=1}^{2M+1} G^{c(v)}_{j} \nonumber\\ &&
	\times
	\left(
		\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{uw,j} }	\left(1 - f^{c(v)}_{w} \right)
		\frac{ f^{c(v)}_{w}  }{ \hat{\tau}^{c(v)}_{wu}  } 	\left(1 - f^{c(v)}_{u,j}\right)
	\right)
  \\
  \label{eq:qd_us}
  \frac{ {\partial}f^{c(v)}_{u,j} }{ {\partial}t } & = & 
	\frac{ f^{c(v)}_{w}  }{ \hat{\tau}^{c(v)}_{wu}  }	\left(1 - f^{c(v)}_{u,j}\right) -
	\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{uw,j} }	\left(1 - f^{c(v)}_{w} \right) \nonumber\\ &&
	+ \frac{ D^{c(v)}_{e} }{ D^{c(v)}_{u} } \left(
		\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eu} } \left(1 - f^{c(v)}_{u,j}\right)
		\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ue} } \left(1 - f^{c(v)}_{e,j}\right)
	\right)  \nonumber\\ &&
	+ \frac{ D^{c(v)}_{g} }{ D^{c(v)}_{u} } \left(
		\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{gu} } \left(1 - f^{c(v)}_{u,j}\right)
		\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ug} } \left(1 - f^{c(v)}_{g,j}\right)
	\right)  \nonumber\\ &&
	- \frac{ \sqrt{ f^{c}_{u,j} f^{v}_{u,j} } }{ \hat{\tau}^{c(v)}_{ur} }
  \\
  \label{eq:qd_es}
  \frac{ {\partial}f^{c(v)}_{e,j} }{ {\partial}t } & = & 
	\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ue} }	\left(1 - f^{c(v)}_{e,j}\right) -
	\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eu} }	\left(1 - f^{c(v)}_{u,j}\right) \nonumber\\ &&
	+ \frac{ D^{c(v)}_{g} }{ D^{c(v)}_{e} } \left(
		\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{ge} } \left(1 - f^{c(v)}_{e,j}\right)
		\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eg} } \left(1 - f^{c(v)}_{g,j}\right)
	\right)  \nonumber\\ &&
	- \frac{ \sqrt{ f^{c}_{e,j} f^{v}_{e,j} } }{ \hat{\tau}_{dr} } - \frac{\Gamma}{N^{c(v)}_{E,j}} \nonumber\\ &&
	\times
	\sum_{k} \frac{ P_{k,in}g^{e}_{jk} \bigl[e^{( [ \Gamma g(t,E_k) - \alpha_{i} ] L_{ca})} - 1 \bigr]}{E_k\bigl[ \Gamma g \left( t,E_k \right) - \alpha_{i} \bigr]} \nonumber\\ &&
	\times
	\left( f^{c}_{e,j} + f^{v}_{e,j} - 1 \right)
  \\
  \label{eq:qd_gs}
  \frac{ {\partial}f^{c(v)}_{g,j} }{ {\partial}t } & = & 
	\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ug} }	\left(1 - f^{c(v)}_{g,j}\right) -
	\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{gu} }	\left(1 - f^{c(v)}_{u,j}\right) \nonumber\\ &&
	\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eg} } \left(1 - f^{c(v)}_{g,j}\right) -
	\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{ge} } \left(1 - f^{c(v)}_{e,j}\right) \nonumber\\ &&
	- \frac{ \sqrt{ f^{c}_{g,j} f^{v}_{g,j} } }{ \hat{\tau}_{dr} } - \frac{\Gamma}{N^{c(v)}_{G,j}} \nonumber\\ &&
	\times
	\sum_{k} \frac{ P_{k,in}g^{g}_{jk} \bigl[e^{( [ \Gamma g(t,E_k) - \alpha_{i} ] L_{ca})} - 1 \bigr]}{E_k\bigl[ \Gamma g \left( t,E_k \right) - \alpha_{i} \bigr]} \nonumber\\ &&
	\times
	\left( f^{c}_{g,j} + f^{v}_{g,j} - 1 \right)
\end{eqnarray}

({\ref{eq:qd_qw}})式から({\ref{eq:qd_gs}})式の左辺は活性層の各層でのキャリア占有率の時間微分を表している．ここで，各式の詳細について以下に述べる．
({\ref{eq:qd_qw}})式は微小時間におけるQWにおけるキャリア占有率の挙動である．右辺各項について説明する．
\begin{description}
  \item[式({\ref{eq:qd_qw}})第1項 $\frac{I}{qN^{c(v)}_{W}}$]\mbox{}\\
	電流によりQWに注入されるキャリアを表す．注入電流$I$[C/s]を電荷素量$q$[C]で割っているのは電流をキャリア数に換算するためであり，キャリア数の上限を超えないように$N^{c(v)}_{W}$で割っている．
  \item[式({\ref{eq:qd_qw}})第2項 $-\frac{ \sqrt{ f^{c}_{w} f^{v}_{w} } }{ \hat{\tau}^{c(v)}_{wr} }$]\mbox{}\\
	非発光再結合により消失するキャリアを表す．非発光再結合とは電子とホールが衝突してエネルギーが光とならず熱として放出する現象のことである．非発光再結合時間$\hat{\tau}^{c(v)}_{wr}$[s]での除算は，単位時間当たりの密度への換算であり，$\hat{\tau}^{c(v)}_{wr}$秒間隔でこの状態が発生することを表す．
  \item[式({\ref{eq:qd_qw}})第3項 $\frac{ D^{c(v)}_{u} }{ D^{c(v)}_{w} }$]\mbox{}\\
	QWに対するUSの縮退係数を表す．
  \item[式({\ref{eq:qd_qw}})第3項 $G^{c(v)}_{j}$]\mbox{}\\
	不均一広がりの内，QDのj番目の存在確率を表す．
  \item[式({\ref{eq:qd_qw}})第3項 $\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{uw,j} }	\left(1 - f^{c(v)}_{w} \right)$]\mbox{}\\
	USからQWへのキャリア脱出を表す．$1 - f^{c(v)}_{w}$ と $f^{c(v)}_{u,j}$ の乗算は，QWに脱出できる$f^{c(v)}_{u,j}$の割合を示している．キャリア脱出時間(US $\to$ QW)$\hat{\tau}^{c(v)}_{uw,j}$[s]での除算は，単位時間当たりの密度への変換であり，$\hat{\tau}^{c(v)}_{uw,j}$秒間隔でこの状態が発生することを表している．
  \item[式({\ref{eq:qd_qw}})第3項 $-\frac{ f^{c(v)}_{w}  }{ \hat{\tau}^{c(v)}_{wu}  } 	\left(1 - f^{c(v)}_{u,j}\right)$]\mbox{}\\
	QWからUSへのキャリア緩和を表す．$1 - f^{c(v)}_{u,j}$ と $f^{c(v)}_{w}$ の乗算は，USに遷移できる$f^{c(v)}_{w}$の割合を示している．キャリア緩和時間(QW $\to$ US)$\hat{\tau}^{c(v)}_{wu}$[s]での除算は，単位時間当たりの密度への変換であり，$\hat{\tau}^{c(v)}_{wu}$秒間隔でこの状態が発生することを表している．
\end{description}

\begin{eqnarray}

\subsection{QDSOAによる利得}
QDSOAの単位長さあたりの利得$g(t,E_k)$は
\begin{equation}
  \label{eq:gain}
  g(t,E_k) = \sum_{l=e,g}\sum_{j=1}^{2M+1} g^{l}_{j,k}(E_k)\bigl[f^{c}_{l,j}+f^{v}_{l,j}-1\bigr]
\end{equation}
で表される．これは，単位長さあたりの利得$g(t,E_k)$がES，GSのキャリア占有率に依存して変化することを示している．\\
また，j番目のES，GS層それぞれが波長kの光波に与える利得$g^{e(g)}_{j,k}$は
\begin{equation}
  g^{e(g)}_{j,k} = \frac{ \pi \hbar q^{2} N_{D} D_{e(g)} }{ c n_{r} \varepsilon_{0} m^{2}_{0} E_{k} } \left|M_{env}\right|^{2} \left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> L^{e(g)}_{j} G^{s}_{j}
\end{equation}
で表される．ここで，$L^{e(g)}_{j}$，$G^{s}_{j}$はそれぞれ光子エネルギーの均一広がり，j番目のQD層の放射再結合の確率を表しており，
\begin{eqnarray}
  L^{e(g)}_{j}  	& = & \frac{1}{\pi} \frac{ \Gamma_{h} / 2 }{\left(E^{s}_{e(g),j} - E_k\right)^{2} + \left(\Gamma_{h}/2\right)^{2}} \\
  G^{s}_{j}  	& = & \frac{1}{\sqrt{2 \pi \sigma^{2}_{in,s}}} \exp \left[-\frac{\left(E^{s}_{g,j}-E^{s}_{g,M}\right)^{2}}{2\sigma^{2}_{in,s}}\right] \Delta E
\end{eqnarray}
と表される．また$E^{s}_{u(e,g),j}$はQD各層におけるエネルギーバンドギャップであり，
\begin{equation}
  E^{s}_{u(e,g),j} = E^{c}_{u(e,g),j} - E^{v}_{u(e,g),j}
\end{equation}
となる．このとき，出力光強度$P_{k,out}$は増幅利得を$G(t,E_k)$として入力光強度を$P_{k,in}$とすると
\begin{equation}
  P_{k,out} = G(t,E_k)P_{k,in}
\end{equation}
と表される．また，増幅利得は閉じ込め係数を$\Gamma$，損失係数を$\alpha_{i}$，活性層の長さを$L_{ca}$として({\ref{eq:total_gain}})式で表される．
\begin{equation}
  \label{eq:total_gain}
  G(t,E_k) = \exp\left(\left[\Gamma g(t,E_k) - \alpha_{i}\right]L_{ca}\right)
\end{equation}

\subsection{QDSOAによる位相変化}
QDSOAはバンド充填効果，自由電子吸収効果により屈折率変化が起こることにより光波の位相変化が起きる{\cite{bf_and_fca_phase}}．バンド充填効果による屈折率変化はQW層からGS層までそれぞれ
\begin{eqnarray}
  \Delta n^{w}_{ib} (t,E_k) & = & 
	\frac
	{-\Gamma_{w} \pi \hbar^{2} q^{2} N_{D}}
	{2 n_{r} \varepsilon_{0} m^{2}_{0} E^{2}_{k}}
	\left|M_{env}\right|^{2} \left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> \nonumber\\ &&
  	\times 
	\frac
	{D^{c}_{w} D^{v}_{w}}
	{D^{c}_{w} + D^{v}_{w}} 
	B_{w}\left[f^{c}_{w}+f^{v}_{w}-1\right]
  \\
  \Delta n^{u}_{ib} (t,E_k) & = & 
	\sum_{j=1}^{2M+1} 
	\frac
	{-\Gamma \pi \hbar^{2} q^{2} N_{D}}
	{2 n_{r} \varepsilon_{0} m^{2}_{0} E^{2}_{k}} 
	\left|M_{env}\right|^{2}
	\left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> \nonumber\\ &&
  	\times \frac
	{ D^{c}_{u} D^{v}_{u} }
	{D^{c}_{u} + D^{v}_{u}}
	G^{s}_{j}B_{u,j}\left[f^{c}_{u,j}+f^{v}_{u,j}-1\right]
  \\
  \Delta n^{e(g)}_{ib} (t,E_k) & = &
	\sum_{j=1}^{2M+1} 
	\frac
	{-\Gamma \pi \hbar^{2} q^{2} N_{D} D_{e(g)} }
	{2 n_{r} \varepsilon_{0} m^{2}_{0} E^{2}_{k}}
	\left|M_{env}\right|^{2}
	\left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> \nonumber\\ &&
  	\times G^{s}_{j} B_{e(g),j}	\left[f^{c}_{e(g),j}+f^{v}_{e(g),j}-1\right]
\end{eqnarray}
と表される．ここで$B_{w}$，$B_{u(e,g),j}$は屈折率における均一広がりを表し，以下の式で与えられる．
\begin{eqnarray}
  B_{w} 	    & = & \frac{\left(E^{s}_{w} - E_{k}\right) / \pi}{\left(E^{s}_{w} - E_{k}\right)^{2} + \left(\Gamma_{h} / 2\right)^{2}} \\
  B_{u(e,g),j} & = & \frac{\left(E^{s}_{u(e,g),j} - E_{k}\right) / \pi}{\left(E^{s}_{u(e,g),j} - E_{k}\right)^{2} + \left(\Gamma_{h} / 2\right)^{2}}
\end{eqnarray}
自由電子吸収効果による屈折率変化の式は以下のようになる．
\begin{eqnarray}
  \Delta n^{w}_{fc} (t,E_k) & = & - \frac{ \Gamma_{w} \hbar^{2} q^{2} }{ 2 V_{w} n_{r} \varepsilon_{0} E^{2}_{k} } 
  \left(
	\frac{ N^{c}_{W} }{ m^{*_{c}}_{w} } f^{c}_{w} + 
	\frac{ N^{v}_{W} }{ m^{*_{v}}_{w} } f^{v}_{w}
  \right) 
  \\
  \Delta n^{u(e,g)}_{fc} (t,E_k) & = & - \frac{ \Gamma \hbar^{2} q^{2} }{ 2 V_{d} n_{r} \varepsilon_{0} E^{2}_{k} } \nonumber\\ &&
  \left( \sum_{j=1}^{2M+1} 
	  \frac{ f^{c}_{u(e,g),j} N^{c}_{U(E,G),j} }{ m^{*_{c}}_{d} } 
	+\frac{ f^{v}_{u(e,g),j} N^{v}_{U(E,G),j} }{ m^{*_{v}}_{d} }
  \right)
\end{eqnarray}
バンド充填効果，自由電子吸収効果による屈折率変化量の和を$\Delta n(t,E_k)$とすると位相変化量は
\begin{equation}
  \label{eq:phase_diff}
  \Delta \phi (t,E_k) = -\frac{2 \pi L_{ca}}{\lambda_{k}} \Delta n(t,E_k)
\end{equation}
と表される．

\subsection{光波の伝搬方程式}
伝搬方程式とは光波が進行方向に進んだ時の振幅変化と位相変化を導出するもので，光波の伝搬方程式は単位距離あたりの増幅利得を$d G(t,E_{k})$，位相変化量を$d \phi (t,E_{k})$とすると
\begin{equation}
  \label{eq:denpan}
  \frac{{\partial}E}{{\partial}z} = E\left(\frac{d G(t,E_{k})}{2}+j d \phi (t,E_{k})\right)
\end{equation}
で表せる．$z$は光波の進行方向を表しており，$j$は虚数単位を表している．
光が伝搬方向に沿って$\Delta z$だけ進んだときの電場は$z$の位置の電場を$E(z)$とすると
\begin{equation}
  \label{eq:denpan_opened}
  E(z+\Delta z) = E(z)\exp\left(dG(t,E_{k})+jd\phi (t,E_{k})\right) \delta z
\end{equation}
と表される．

\subsection{非線形状態空間法(Nonlinear State Space Method:NSSM)}
伝搬方程式を解くには適切な単位距離を設定して各セクションごとに電場を求めるTransfer Matrix Method(TMM)と呼ばれる手法が存在する．しかし，電子計算機上での計算量が多いことから本研究では計算量を削減できるNSSMと呼ばれる手法を用いる．NSSMは各セクションごとの電場を求めることなく入力された電場から出力される電場を算出する方法であり，TMMと同様の結果を得られることが知られている{\cite{NSSM1,NSSM2}}．NSSMを用いると式(\ref{eq:total_gain})，式(\ref{eq:phase_diff})よりQDSOAに入力する光の電場を$E(0)$とするとQDSOAから出力される光の電場$E(L_{ca})$は(\ref{eq:electric_field_out})式より求められる．
\begin{equation}
  \label{eq:electric_field_out}
  E(L_{ca}) = E(0)\exp\left(G(t,E_{k})+j\Delta \phi (t,E_{k})\right)
\end{equation}

\section{フォトニック結晶(Photonic Crystal:PC)}
PCとは屈折率の異なる物質が周期的に並んだナノ周期の構造体であり，1次元から3次元の結晶が構築可能である．自然界の蝶の羽や宝石の一種が，見る方向によって光沢が変わることが知られているが，これは自然界で形成されたナノ周期構造による光の侵入阻止効果によるものであり，PCと同等の原理である．ナノ周期構造を人工的に，かつ二次元または三次元的に作成することは極めて困難だったがナノテクノロジー技術の発展により実現可能となった{\cite{photonic_crystal1, photonic_crystal2}}．構造の周期を使用する光の波長に応じて変化させることにより特定の波長域の光が結晶中で伝搬することを禁止でき，その特性から人為的に欠陥構造を作ることにより特定の波長の光のみを伝搬させることができる．これは光の回折，散乱，干渉による影響によるもので，その特性を利用した光ファイバ，光導波路，光メモリが研究されている．

\section{分散関係}
分散関係は波数と角振動数の関係を表し，分散がある場合は波数$k$に対して一定の角振動数$\omega(k)$が成立する．分散がない場合では，波数と角振動数には常に比例関係$\omega(k) = vk$が成立する．例えば，真空中であれば分散がないので真空中の光速を$c_{0}$とすると分散関係$\omega(k) = c_{0}k$が成り立つ．

\section{スローライト効果}
スローライト効果とは媒質中で光波の伝搬速度が遅くなる現象のことで電磁波の位相速度と群速度がずれることにより発生する．位相速度とは光波の腹や節の位置が移動する速度のことであり，群速度とは様々な波数を持つ波の集まりである波束が移動する速度のことである．情報の``0''，``1''を光で伝搬する際には様々な波数を持つ波の集まりである光パルスを用いることから波長によって位相速度が異なると，波の強め合う位置がずれることによりパルスの進みが遅くなるため，スローライト効果が発生する．
位相速度$v_{p}$は
\begin{equation}
  v_{p} = \frac{\partial x}{\partial t} = \frac{\omega(k)}{k}
\end{equation}
と表され，群速度$v_{g}$は
\begin{equation}
  v_{g} = \frac{\partial \omega(k)}{\partial k}
\end{equation}
と表される．このとき，スローダウン要素は$S(\lambda)=v_{p}/v_{g}$と表される{\cite{slow_light1,slow_light2}}．

\section{フォトニック結晶導波路(Photonic Crystal Waveguides:PCW)}
2次元のPCは屈折率が一様な半導体に対して周期的に空孔を設けることで得られる結晶構造であり，実用的なデバイスとしてはPCWが挙げられる。PCWのモデルを図{\ref{fig:model_of_pcw}}に示す．PCには特定の帯域の光を結晶内部に伝搬させない性質があり，この帯域はフォトニックバンドギャップと呼ばれる．PCWはこの性質を利用しており，モデル図のように1列分の欠陥を作ることで欠陥部分をフォトニックバンドギャップ帯域の光のみが伝搬できるようにしている{\cite{photonic_crystal2}}．また，使用する帯域の光の波長をもとに空孔の間隔を調整することで光の速度を調整することができるが，構造によっては高次の分散が発生し，パルスの広がりや変形が起きるため適切な間隔に調整する必要がある{\cite{slow_light1}}．

\section{PC-QDSOAの入出力特性の解析}
ここでは，PC-QDSOAの入出力特性の解析に用いる手法，式について説明する．

\subsection{誘電体の屈折率}
誘電体の屈折率は構成する媒体の屈折率の平均から求めることができ，屈折率の平均は媒体の体積比によって決まる．PC-QDSOAはGaAs，${\rm In_{0.15}Ga_{0.85}As }$の屈折率及び体積比から求めることができる．また，媒体の屈折率は波長依存性をもち，GaAsの屈折率$n_{\rm GaAs}(\lambda)$は波長$\lambda$に対して
\begin{equation}
  n_{\rm GaAs}(\lambda) = \left[10.906 + \frac{0.97501}{\lambda^{2}-0.2797}-0.002467\lambda^{2}\right]^{1/2}
\end{equation}
と表される{\cite{GaAsIndex}}．
${\rm In_{x}Ga_{1-x}As }$の屈折率$n{\rm In_{x}Ga_{1-x}As }(\lambda,x)$は波長$\lambda$及びインジウムの構成比$x$に対して
\begin{equation}
  n{\rm In_{x}Ga_{1-x}As }(\lambda,x) = \sqrt{8.950+\frac{ 2.054 }{ 1-\left(\frac{0.6254 E_{GaAs}}{\lambda E_{g}(x)}\right) }}
\end{equation}
と表される{\cite{InGaAsIndex}}．また，${\rm In_{x}Ga_{1-x}As }$のバンドギャップエネルギー$E_{g}(x)$はGaAsのエネルギーバンドギャップを${\rm E_{g_{GaAs}}}$として(\ref{eq:InGaAs_bandgap})式より求めることができる．
\begin{equation}
  \label{eq:InGaAs_bandgap}
  E_{g}(x) = E_{g_{GaAs}} - (1.501)x + (0.436)x^{2}
\end{equation}
ここで，PC-QDSOAにおけるGaAsの割合を$V_{\rm GaAs}$，${\rm In_{x}Ga_{1-x}As}$の割合を$V_{\rm In_{x}Ga_{1-x}As}$とすると誘電体の屈折率$n_{core}(\lambda)$は
\begin{equation}
  n_{core}(\lambda) = \frac
	{n_{\rm GaAs}(\lambda) V_{\rm GaAs} + n{\rm In_{x}Ga_{1-x}As }(\lambda,x) V_{\rm In_{x}Ga_{1-x}As}}
	{V_{\rm GaAs} + V_{\rm In_{x}Ga_{1-x}As}}
\end{equation}
と表される．

\subsection{実効屈折率}
屈折率が一様でない媒体を解析するには実効屈折率を用いる．フォトニック結晶導波路には光の伝搬方向に関する実効屈折率を用いる{\cite{effective_index_approximations}}．

\subsection{PC-QDSOAのレート方程式}
PC-QDSOAのキャリア密度に関する解析にはQDSOAと同様にレート方程式を用いる．本研究に用いるPC-QDSOAのレート方程式を式に示す{\cite{pc_qd_soa_rate_equation}}．また，各項の意味はQDSOAと同等である．

\begin{eqnarray}
  \label{eq:pc_qd_qw}
  \frac{ {\partial}f^{c(v)}_{w} }{ {\partial}t } & = & 
	\frac{I}{qN^{c(v)}_{W}} - 
	\frac{ \sqrt{ f^{c}_{w} f^{v}_{w} } }{ \hat{\tau}^{c(v)}_{wr} } + 
	\frac{ D^{c(v)}_{u} }{ D^{c(v)}_{w} } \sum_{j=1}^{2M+1} G^{c(v)}_{j} \nonumber\\ &&
	\times
	\left(
		\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{uw,j} }	\left(1 - f^{c(v)}_{w} \right)
		\frac{ f^{c(v)}_{w}  }{ \hat{\tau}^{c(v)}_{wu}  } 	\left(1 - f^{c(v)}_{u,j}\right)
	\right)
  \\
  \label{eq:pc_qd_us}
  \frac{ {\partial}f^{c(v)}_{u,j} }{ {\partial}t } & = & 
	\frac{ f^{c(v)}_{w}  }{ \hat{\tau}^{c(v)}_{wu}  }	\left(1 - f^{c(v)}_{u,j}\right) -
	\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{uw,j} }	\left(1 - f^{c(v)}_{w} \right) \nonumber\\ &&
	+ \frac{ D^{c(v)}_{e} }{ D^{c(v)}_{u} } \left(
		\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eu} } \left(1 - f^{c(v)}_{u,j}\right)
		\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ue} } \left(1 - f^{c(v)}_{e,j}\right)
	\right)  \nonumber\\ &&
	+ \frac{ D^{c(v)}_{g} }{ D^{c(v)}_{u} } \left(
		\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{gu} } \left(1 - f^{c(v)}_{u,j}\right)
		\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ug} } \left(1 - f^{c(v)}_{g,j}\right)
	\right)  \nonumber\\ &&
	- \frac{ \sqrt{ f^{c}_{u,j} f^{v}_{u,j} } }{ \hat{\tau}^{c(v)}_{ur} }
  \\
  \label{eq:pc_qd_es}
  \frac{ {\partial}f^{c(v)}_{e,j} }{ {\partial}t } & = & 
	\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ue} }	\left(1 - f^{c(v)}_{e,j}\right) -
	\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eu} }	\left(1 - f^{c(v)}_{u,j}\right) \nonumber\\ &&
	+ \frac{ D^{c(v)}_{g} }{ D^{c(v)}_{e} } \left(
		\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{ge} } \left(1 - f^{c(v)}_{e,j}\right)
		\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eg} } \left(1 - f^{c(v)}_{g,j}\right)
	\right)  \nonumber\\ &&
	- \frac{ 1 }{ N^{c(v)}_{E,j} } \sum_{k} \frac
		{\Gamma_{k}P_{k,in}g^{e}_{j,k}\left[e^{\left(\left[g_{mod}(t,\lambda_{k}) - \alpha(\lambda_{k})\right]L_{ca}\right)} - 1\right]}
		{\hbar \omega_{k}\left[g_{mod}(t,\lambda_{k}) - \alpha(\lambda_{k})\right]} \nonumber\\ &&
	\times
	\left( f^{c}_{e,j} + f^{v}_{e,j} - 1 \right)
  \\
  \label{eq:pc_qd_gs}
  \frac{ {\partial}f^{c(v)}_{g,j} }{ {\partial}t } & = & 
	\frac{ f^{c(v)}_{u,j} }{ \hat{\tau}^{c(v)}_{ug} }	\left(1 - f^{c(v)}_{g,j}\right) -
	\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{gu} }	\left(1 - f^{c(v)}_{u,j}\right) \nonumber\\ &&
	\frac{ f^{c(v)}_{e,j} }{ \hat{\tau}^{c(v)}_{eg} } \left(1 - f^{c(v)}_{g,j}\right) -
	\frac{ f^{c(v)}_{g,j} }{ \hat{\tau}^{c(v)}_{ge} } \left(1 - f^{c(v)}_{e,j}\right) \nonumber\\ &&
	- \frac{ \sqrt{ f^{c}_{g,j} f^{v}_{g,j} } }{ \hat{\tau}_{dr} } - \frac{1}{N^{c(v)}_{G,j}} \nonumber\\ &&
	\times \left(\sum_{k} 
	\frac
		{\Gamma_{k}P_{k,in}g^{g}_{j,k}\left[e^{\left(\left[g_{mod}(t,\lambda_{k}) - \alpha(\lambda_{k})\right]L_{ca}\right)} - 1\right]}
		{\hbar \omega_{k}\left[g_{mod}(t,\lambda_{k}) - \alpha(\lambda_{k})\right]} + \right. \nonumber\\ && 
	\left.
	\frac
		{\Gamma_{probe}P_{probe}g^{e}_{j,probe}\left[e^{\left(\left[g_{mod}(t,\lambda_{probe}) - \alpha(\lambda_{probe})\right]L_{ca}\right)} - 1\right]}
		{\hbar \omega_{probe}\left[g_{mod}(t,\lambda_{probe}) - \alpha(\lambda_{probe})\right]}
	\right) \nonumber\\ &&
	\times
	\left( f^{c}_{g,j} + f^{v}_{g,j} - 1 \right)
\end{eqnarray}

\subsection{PC-QDSOAにおける閉じ込め係数}
半導体レーザは光ファイバと同様に光を閉じ込める部分の屈折率を大きくして，その周りの屈折率を小さくした構造にすることで光を閉じ込めて効率的に光が増幅されるようにしている．PC-QDSOAでは縦方向に関してはエアーギャップを設けることにより光を閉じ込めており，横方向に関してはPCWのフォトニックバンドギャップを利用して光を閉じ込めている．垂直方向に関する波数を$k_{z}$，活性層の厚さを$2d$とすると超越方程式
\begin{equation}
  \tan(k_{z}d) = \sqrt{\frac
	{\omega^{2} \mu_{0} (\varepsilon_{core}-\varepsilon_{cladding}) d^{2}}
	{(k_{z}d)^{2}} - 1}
\end{equation}
から光が伝搬する平面に対して垂直な方向に対する光閉じ込め係数$\Gamma_{z}$は
\begin{equation}
  \Gamma_{z} = \left[1 + \frac
	{\cot(k_{z}d/2)\cos^{2}(k_{z}d/2)}
	{(k_{z}d/2)(1+\sin(k_{z}d)/(k_{z}d))}\right]^{-1}
\end{equation}
より求めることができる．光が伝搬する平面に対して水平な方向に対する光閉じ込め係数$\Gamma_{xy}$は導波路幅を$W$，格子定数を$a$として
\begin{equation}
  \Gamma_{xy} = \frac
	{\int_{-W/2}^{W/2} \int_{-a/2}^{a/2} \frac{1}{2}Re\{E_{t} \times H_{z}^{*}\}.\hat{x}dxdy}
	{\iint_{super cell}\frac{1}{2}Re\{E_{t} \times H_{z}^{*}\}.\hat{x}dxdy}
\end{equation}
より求めることができる．以上が導波路に対する閉じ込め係数$\Gamma_{PCW} = \Gamma_{xy} \Gamma_{z}$であり，QD部に影響する閉じ込め係数$\Gamma(\lambda)$は割合は体積の比率$\xi_{QD} = l_{D}L_{H}/2d$から求めることができ，
\begin{equation}
  \Gamma(\lambda) = \Gamma_{PCW} \xi_{QD}
\end{equation}
と表される．

\subsection{PC-QDSOAにおける損失係数}
PCWでは光が伝搬する平面に対して垂直な方向に光が散乱することによる損失，結晶部分での光の反射，回折により伝搬方向とは逆方向に進む光による損失が発生する．また，SOAではキャリアが光子のエネルギーを吸収することにより損失が発生する．これらのことからPC-QDSOAでは散乱による損失，吸収による損失が発生する．
散乱による損失を$\alpha_{scattering}(\lambda)$，吸収による損失を$\alpha_{absorption}(\lambda)$とすると損失係数はそれらの和$\alpha(\lambda) = \alpha_{scattering}(\lambda) + \alpha_{absorption}(\lambda)$で表される．
また，$\alpha_{scattering}(\lambda)$，$\alpha_{absorption}(\lambda)$はそれぞれ
\begin{eqnarray}
  \alpha_{scattering}(\lambda)  & = & \alpha_{intrinsic}S(\lambda) + \alpha_{extrinsic}S^{2}(\lambda) \\
  \alpha_{absorption}(\lambda) & = & \Gamma_{z}S(\lambda)\left[ \Gamma_{xy}\alpha_{core} + (1-\Gamma_{xy})\alpha_{cladding} \right]
\end{eqnarray}
と求められる．また，自由正孔，自由電子の断面積をそれぞれ$\sigma_{p}$，$\sigma_{n}$とすると$\alpha_{core}$，$\alpha_{cladding}$はそれぞれ
\begin{eqnarray}
  \alpha_{core} & \approx & \alpha_{FCA} = \sigma_{p}N_{A,core} \\
  \alpha_{cladding}  & \approx & \frac{1}{2}\left( \sigma_{p}N_{A,cladding} + \sigma_{n}N_{D,cladding} \right)
\end{eqnarray}
となる．また，キャリアの断面積は自由正孔，自由電子の束縛エネルギーを$\Delta g_{p(n)}$とすると
\begin{equation}
  \sigma_{p(n)} = \frac{q^{2}}{c \varepsilon_{0} n_{r} m^{*_{v(c)}}_{d}\left( \omega_{t} - \Delta g_{p(n)} / \hbar \right)^{2}\tau_{c}}
\end{equation}
と求められる{\cite{loss_coefficient}}．

\subsection{PC-QDSOAによる利得}
PC-QDSOAの単位長さあたりの利得$g_{mod}(t,\lambda_{k})$はスローライト効果，閉じ込め係数の波長依存性を考慮して
\begin{equation}
  g_{mod}(t,\lambda_{k}) = S(\lambda_{k})\Gamma(\lambda_{k})g_{mat}(t,\lambda_{k})
\end{equation}
と表せる．また，j番目のES，GS層それぞれが波長kの光波に与える利得$g^{e(g)}_{j,k}$は
\begin{equation}
  g^{e(g)}_{j,k} = \frac{ \pi \hbar q^{2} N_{D} D_{e(g)} }{ c n_{r} \varepsilon_{0} m^{2}_{0} E_{k} } \left|M_{env}\right|^{2} \left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> L^{e(g)}_{j} G^{s}_{j}
\end{equation}
で表される．ここで，$L^{e(g)}_{j}$，$G^{s}_{j}$はそれぞれ光子エネルギーの均一広がり，j番目のQD層の放射再結合の確率を表しており，
\begin{eqnarray}
  L^{e(g)}_{j}  	& = & \frac{1}{\pi} \frac{ \Gamma_{h} / 2 }{\left(E^{s}_{e(g),j} - E_k\right)^{2} + \left(\Gamma_{h}/2\right)^{2}} \\
  G^{s}_{j}  	& = & \frac{1}{\sqrt{2 \pi \sigma^{2}_{in,s}}} \exp \left[-\frac{\left(E^{s}_{g,j}-E^{s}_{g,M}\right)^{2}}{2\sigma^{2}_{in,s}}\right] \Delta E
\end{eqnarray}
と表される．
このとき，出力光強度$P_{k,out}$は増幅利得を$G(t,E_{k})$として入力強度を$P_{k,in}$とすると
\begin{equation}
  P_{k,out} = G(t,E_k)P_{k,in}
\end{equation}
と表される．また，PCW内で発生する分散，吸収損失とは別にPCWに光が入力される際，PCWから光が出力される際には結合損失が発生する{\cite{model_of_pc_qdsoa}}．そのため，増幅利得$G(t,E_{k})$は活性層の長さを$L_{ca}$，結合損失係数を$\alpha_{cl}$として
\begin{equation}
  \label{eq:pc_total_gain}
  G(t,E_k) = \exp\left(\left[g_{mod}(t,\lambda_{k}) - \alpha(\lambda_{k})\right]L_{ca}\right) - \alpha_{cl}
\end{equation}
と表される．

\subsection{PC-QDSOAによる位相変化}
PC-QDSOAはバンド充填効果，自由電子吸収効果により屈折率変化が起き，それにより光の位相変化が起きる．以下にバンド充填効果による屈折率変化の式を示す．
\begin{eqnarray}
  \Delta n^{w}_{ib} (t,\lambda_{k}) & = & -S(\lambda_{k})\Gamma_{w}(\lambda_{k}) 
	\frac
	{\pi \hbar^{2} q^{2} N_{D}}
	{2 n_{r} \varepsilon_{0} m^{2}_{0} E^{2}_{k}} \nonumber\\ &&
	\left|M_{env}\right|^{2} \left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> 
  	\frac
	{ D^{c}_{w} D^{v}_{w} }
	{D^{c}_{w} + D^{v}_{w}}
	B_{w}\left[f^{c}_{w}+f^{v}_{w}-1\right]
  \\
  \Delta n^{u}_{ib} (t,\lambda_{k}) & = & -S(\lambda{k})\Gamma(\lambda_{k}) \sum_{j=1}^{2M+1} 
	\frac
	{\pi \hbar^{2} q^{2} N_{D}}
	{2 n_{r} \varepsilon_{0} m^{2}_{0} E^{2}_{k}} \nonumber\\ &&
	\times \left|M_{env}\right|^{2} \left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right>
	\frac
	{D^{c}_{u} D^{v}_{u}}
	{D^{c}_{u} + D^{v}_{u}}
	G^{s}_{j}B_{u,j}\left[f^{c}_{u,j}+f^{v}_{u,j}-1\right]
  \\
  \Delta n^{e(g)}_{ib} (t,\lambda_{k}) & = & -S(\lambda{k})\Gamma(\lambda_{k}) \sum_{j=1}^{2M+1} 
	\frac
	{\pi \hbar^{2} q^{2} N_{D} D_{e(g)}}
	{2 n_{r} \varepsilon_{0} m^{2}_{0} E^{2}_{k}} \nonumber\\ &&
	\times \left|M_{env}\right|^{2} \left<\left|\hat{e} \cdot p_{cv}\right|^{2}\right> 
	G^{s}_{j}B_{e(g),j}\left[f^{c}_{e(g),j}+f^{v}_{e(g),j}-1\right]
\end{eqnarray}
自由電子吸収効果による屈折率変化の式は以下のようになる．
\begin{eqnarray}
  \Delta n^{w}_{fc} (t,\lambda_{k}) & = &
	- \frac
	{S(\lambda{k})\Gamma_{w}(\lambda_{k}) \hbar^{2} q^{2}}
	{2 V_{w} n_{r} \varepsilon_{0} E^{2}_{k}} 
  \left(
	\frac{ N^{c}_{W} }{ m^{*c}_{w} } f^{c}_{w} + 
	\frac{ N^{v}_{W} }{ m^{*v}_{w} } f^{v}_{w}
  \right) 
  \\
  \Delta n^{u(e,g)}_{fc} (t,\lambda_{k}) & = &
	- \frac
	{S(\lambda{k})\Gamma(\lambda_{k}) \hbar^{2} q^{2}}
	{2 V_{d} n_{r} \varepsilon_{0} E^{2}_{k}} \nonumber\\ &&
  \times
  \left( \sum_{j=1}^{2M+1} 
	  \frac{ f^{c}_{u(e,g),j} N^{c}_{U(E,G),j} }{ m^{*c}_{d} } +
	  \frac{ f^{v}_{u(e,g),j} N^{v}_{U(E,G),j} }{ m^{*v}_{d} }
  \right)
\end{eqnarray}
バンド充填効果，自由電子吸収効果による屈折率変化量の和を$\Delta n(t,E_k)$とすると位相変化量は
\begin{equation}
  \label{eq:phase_diff}
  \Delta \phi (t,E_k) = -\frac{2 \pi L_{ca}}{\lambda_{k}} \Delta n(t,E_k)
\end{equation}
と表される．

\begin{equation}
  ER(dB) = 10 \log \frac{P_{\min}^{1}}{P_{\max}^{0}}
\end{equation}
